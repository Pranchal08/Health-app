Health Prediction Application

Overview

Health Prediction Application is a Flask-based healthcare management system that allows users to store, manage, and analyze patient health records. The application generates health risk predictions based on medical parameters such as Glucose, Haemoglobin, and Cholesterol levels.

Features

- Create, Read, Update, Delete (CRUD) patient records
- Search patient records
- SQLite database integration
- Responsive Bootstrap 5 user interface
- Health risk prediction system
- AI-style health assessment remarks
- Flask and SQLAlchemy backend
- Easy-to-use dashboard

Technology Stack

- Python
- Flask
- SQLAlchemy
- SQLite
- HTML5
- Bootstrap 5
- JavaScript

Run Application

Open Browser

http://127.0.0.1:5000

How It Works

1. User enters patient details.
2. Health parameters are stored in SQLite database.
3. The system analyzes Glucose, Haemoglobin, and Cholesterol values.
4. Health risk remarks are generated automatically.
5. Users can edit, search, and delete records.
